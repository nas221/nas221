## Projects

**[RoleRadar](https://github.com/nas221/roleradar)**
AI-powered job aggregation platform. Scrapes and processes 3,000+ listings/month,
serving 50+ active users. Built with Next.js, Flask, PostgreSQL, and the Gemini API.

Production-style event ticketing REST API. Thread-safe concurrent booking engine
using Java concurrency and optimistic locking — zero double bookings under 100
simultaneous requests. Built with Java 21, Spring Boot, Spring Security, and Hibernate.

**[Distributed Task Platform](https://github.com/nas221/distributed-task-platform)**
Asynchronous task processing system using RabbitMQ message queues and background
workers. JWT auth, Docker containerisation, deployed on AWS.
